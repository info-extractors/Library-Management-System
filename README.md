# Library Management System in C++

## Overview

This project is a console-based Library Management System developed using C++.  
It is designed to demonstrate the practical implementation of:

- Object-Oriented Programming (OOP)
- STL Containers
- Searching Operations
- Modular Programming

The system allows users to manage books inside a library through a menu-driven interface.

---

# Features

- Add new books
- Display all books
- Search books by title
- Issue books
- Return books
- Interactive console-based menu system
- Dynamic book storage using STL vectors

---

# Technologies Used

- C++
- STL (`vector`)
- Object-Oriented Programming

---

# OOP Concepts Implemented

| Concept | Usage |
|---|---|
| Class & Objects | `Book`, `Library` |
| Encapsulation | Private data members |
| Constructors | Book initialization |
| Member Functions | Operations on books |
| Abstraction | Hiding implementation details |
